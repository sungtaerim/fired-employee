# Получение списка уволенных сотрудников, активных в Teamcenter Список включает в себя: - Сотрудника - uid - имя - id - вне офиса - дата последнего входа - полная группа - суррогат - Суррогат - uid - имя - id - начальная дата передачи задания - конечная дата передачи задания

Получение списка уволенных сотрудников из таблицы Spic_uv: http://srv-tcfox:8080/fired-employee-1.0/fired

Получение информации о пользователе Teamcenter: http://srv-tcfox:8080/fired-employee-1.0/user/{user_id} {user_id} - ID пользователя

Получение списка уволенных сотрудников, активных в Teamcenter: http://srv-tcfox:8080/fired-employee-1.0/all?check=yes Параметр check позволяет в отчёт добавить только активных сотрудников, не обязательный параметр
