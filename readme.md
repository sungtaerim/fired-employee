# Получение списка уволенных сотрудников, активных в Teamcenter

### Список включает в себя:
- Сотрудника
    - uid
    - имя
    - id
    - вне офиса
    - дата последнего входа
    - полная группа
    - суррогат
- Суррогат
    - uid
    - имя
    - id
    - начальная дата передачи задания
    - конечная дата передачи задания

### Получение списка уволенных сотрудников из таблицы Spic_uv:

> http://{server_name}:{port}/fired-employee-1.0/fired.{format}

{format} - возвращаемый тип (json или xml)

### Получение информации о пользователе Teamcenter:

> http://{server_name}:{port}/fired-employee-1.0/user/{user_id}.{format}

{user_id} - ID пользователя

{format} - возвращаемый тип (json или xml)

### Получение списка уволенных сотрудников, активных в Teamcenter:

> http://{server_name}:{port}/fired-employee-1.0/all.{format}?status=0

{format} - возвращаемый тип (json или xml)

Параметр status=0 позволяет в отчёт добавить только активных сотрудников, status=1 позволяет добавить в отчёт всех уволенных сотрудников
